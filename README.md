### Utilisation : 

Un make file est livré avec le code, il est excécutable grâce à la commande :

```
make
```
 
Une fois excécuté, un fichier **_exe_** est créé dans le répertoire **_./bin_** :

```
./bin/exe
```

Afin d'excécuter le programme.
